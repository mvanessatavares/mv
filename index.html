Olá, me chamo Vanessa, sou formada em Análise e Desenvolvimento de Sistemas pelo Instituto Federal da Paraíba e atualmente estou cursando uma pós-graduação em Engenharia de Software, reforçando meu compromisso com a educação continuada e o aperfeiçoamento técnico.

Atuo no setor de tecnologia da informação desde 2016, acumulando uma experiência diversificada que inclui desde o design UX/UI até o desenvolvimento completo de softwares, passando pela documentação detalhada e a realização de testes rigorosos para garantir a qualidade e a usabilidade dos sistemas. Tenho experiência em projetos de diferentes níveis de complexidade, trabalhando em colaboração com equipes multifuncionais e adaptando-me a metodologias ágeis para alcançar resultados eficazes.

Minha trajetória é marcada pela busca incessante de conhecimento e inovação. Acredito que, em um setor em constante evolução como o da tecnologia, é essencial sair da zona de conforto e se manter atualizado com as melhores práticas e as tendências do mercado. Com isso, sou uma profissional focada, criativa e comprometida em aplicar habilidades para resolver desafios de forma eficiente e agregar valor a cada projeto em que me envolvo.

Projetos

Esses são meus projetos mais recentes.


PetAdote




PetAdote foi criado com o objetivo de apoiar ONGs e protetores de animais na ampliação de sua visibilidade e alcance, facilitando a divulgação de animais para adoção. O projeto foi desenvolvido utilizando React para a construção da interface visual e Node.js no backend, garantindo uma aplicação moderna e eficiente. O armazenamento dos dados de animais e usuários é feito no banco de dados MongoDB, proporcionando flexibilidade e escalabilidade. O design da aplicação foi concebido no Figma, garantindo uma experiência de usuário intuitiva e agradável.

Repositorio do projeto


DivulgaEventos




O DivulgaEvento foi desenvolvido como uma plataforma voltada para a divulgação de eventos acadêmicos, proporcionando um espaço dedicado para promover e compartilhar informações. A aplicação foi construída utilizando TypeScript e o Vite, garantindo um desenvolvimento ágil e eficiente. Para o design, foram utilizadas as ferramentas Figma e Photoshop, assegurando uma interface visual moderna e atraente.

Repositorio do projeto

ReciclaAE



O ReciclaAE é um sistema móvel desenvolvido para localizar os pontos de coleta mais próximos de cada usuário. Foi criado utilizando o React Native e toda a sua interface foi prototipada no Figma.

Repositorio do projeto

Carona




O Sistema de Carona foi projetado para oferecer aos usuários uma maneira conveniente de solicitar um veículo. A aplicação foi desenvolvida utilizando React.js e TypeScript, garantindo uma interface dinâmica e de alta performance. O design da plataforma foi elaborado no Figma, assegurando uma experiência de usuário intuitiva e moderna.
